Movavi Project Converter

Описание проекта

Movavi Project Converter — это инструмент для конвертации проектов Movavi между различными версиями. Быстро открывайте старые проекты или подготавливайте их для новых версий программы.

Как использовать

1. Скачайте и распакуйте конвертер.

2. Выберите проект Movavi (.mepj).

3. Выберите целевую версию и нажмите Convert.
